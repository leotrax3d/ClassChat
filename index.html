classchat

classchat is a lightweight local chat application built for school networks, shared folders and offline communication.

no server. no accounts. no installation. just a shared folder and a simple interface that lets everyone join the conversation.

channels, online status, file attachments, typing indicators and user presence are built in. everything runs through files stored in a shared directory.

made for classrooms, labs and small local networks where a fast and simple chat solution is needed.

05

getting started

1. download
download the latest release and extract the zip file anywhere on your computer.
step
2. run start.bat
launch start.bat. the batch file automatically starts the powershell interface and loads classchat.
step
3. select shared folder
choose a shared network folder that all users can access. classchat will create the required files and folders automatically.
step
4. start chatting
join a channel, choose your name and send messages. users connected to the same shared folder will see each other instantly.
done
